## Description
This is a sample boilerplate repository that contains  [NestJS](https://docs.nestjs.com/) and  [NextJS](https://nextjs.org/docs)

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode for both nestJS and nextJS
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
